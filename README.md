# Development setup

## Run the saleor server

1. Browse your `saleor` project folder
2. Run the following commands

```
pipenv shell
python ./manage.py runserver
```

## Reset the database data

1. Browse your `saleor` project folder
2. Run the following commands

```
pipenv run ./resetDB.sh
```

## Run the frontend

1. Run `quasar dev`
